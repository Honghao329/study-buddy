# 学习伴侣 — 首页（"今天"）设计文档

## 一、产品定位

学习伴侣是一个"学习自律 + 伙伴监督"小程序。首页是用户每天打开的第一个页面，核心任务是回答一个问题：**"今天我学了吗？"**

## 二、目标用户

大学生 / 考研党 / 自学者，需要每日记录学习、和伙伴互相监督。

## 三、设计原则

1. **一屏一焦点** — 打开就知道今天该做什么
2. **Forest/Duolingo 模式** — 主 CTA 只有一个
3. **不重复 Tab 栏** — Tab 栏已有"打卡""社区""我的"，首页不再放这些入口
4. **内容驱动** — 不是空荡荡的仪表盘，有内容（笔记/动态）让人想刷

## 四、技术栈

- 微信小程序原生
- Vant Weapp 组件库（项目已集成）
- 使用系统导航栏（`navigationBarTitleText: "今天"`），不要自定义导航栏
- 导航栏白底黑字：`navigationBarBackgroundColor: "#ffffff"`, `navigationBarTextStyle: "black"`

## 六、各区域详细规格

### 6.1 签到状态条

- **位置**：页面最顶部，紧贴导航栏下方
- **结构**：一个白色圆角卡片，左侧有 6rpx 的彩色竖线
- **左半部分**：`🔥 {streak}天连续 · 累计{totalDays}天`
- **右半部分**：
  - 未签到：蓝色按钮"签到"，点击弹出签到弹窗
  - 已签到：绿色文字"✓ 已签"，点击跳转到签到日历页
- **竖线颜色**：未签到 = `#FF9800`（橙），已签到 = `#4CAF50`（绿）
- **样式**：`border-radius: 14rpx; padding: 18rpx 24rpx; margin: 12rpx 24rpx;`

### 6.2 今日打卡任务

- **条件**：用户有参与的打卡任务时显示
- **标题行**：左"今日打卡"，右"{done}/{total}"蓝色
- **每个任务一张独立白色卡片**：
  - 左边：圆形 checkbox（30rpx），未完成空心灰边框，完成后绿色填充+白色✓
  - 中间：任务名（28rpx 加粗）+ 下面一行小字"X人参与"（22rpx 灰色）
  - 右边：箭头图标
  - 已完成的任务名加删除线，颜色变灰
- **点击**：跳转打卡详情页
- **间距**：卡片间 `margin-bottom: 10rpx`
- **无任务时**：显示 Vant Empty 组件 + "去看看"按钮

### 6.3 最新动态

- **标题**：左对齐"最新动态"，28rpx 加粗
- **容器**：一个白色圆角卡片包裹所有动态
- **每条动态**：
  - 头像（30rpx 圆形）+ 文字描述（24rpx）+ 时间（20rpx 灰色）
  - 用 `border-bottom: 1rpx solid #f8f8f8` 分割
- **数据来源**：`GET /api/home/activity`

### 6.4 热门笔记

- **标题行**：左"热门笔记"，右"更多"（点击跳转社区 Tab）
- **每篇笔记一张独立白色卡片**：
  - 第一行：头像（36rpx）+ 作者名
  - 第二行：标题（28rpx 加粗，最多2行）
  - 第三行：正文预览（24rpx 灰色，最多3行）
  - 第四行：点赞数 + 浏览数 + 评论数（22rpx 灰色图标）
- **点击**：跳转笔记详情页
- **数据来源**：`GET /api/note/public_list?page=1&size=10&sort=hot`

## 七、签到弹窗

- **触发**：点击签到状态条的"签到"按钮
- **组件**：Vant Popup，`position="bottom" round`
- **表单字段**：
  1. 学习时长（分钟）— number input，选填
  2. 学习状态 — 三选一横排按钮：😊高效 / 😐一般 / 😴疲惫
  3. 学习内容 — textarea，选填，最多200字
- **提交**：`POST /api/sign/do { duration, status, content }`
- **成功后**：关闭弹窗，刷新页面数据

## 八、数据接口

| 接口 | 用途 |
|------|------|
| `GET /api/sign/stats` | 签到统计（streak, totalDays, totalDuration, todaySigned） |
| `GET /api/checkin/list?page=1&size=10` | 打卡任务列表 |
| `GET /api/checkin/today_done_ids` | 今日已完成的打卡任务ID数组 |
| `GET /api/home/activity` | 最新动态列表 |
| `GET /api/note/public_list?page=1&size=10&sort=hot` | 热门笔记 |
| `GET /api/message/unread_count` | 未读消息数 |
| `POST /api/sign/do` | 执行签到 |

## 十、交互细节

1. 签到按钮点击后弹出 bottom popup，不跳页
2. 已签到后按钮变为"✓ 已签"绿色文字，点击跳转到签到日历页（`/pages/sign/sign`）
3. 打卡任务卡片点击跳转详情页
4. 笔记卡片点击跳转详情页
5. 支持下拉刷新
6. 所有白色卡片 hover 状态：`background: #fafbfc`

## 十一、注意事项

1. **不要自定义导航栏** — 用系统默认导航栏，稳定且不会有适配问题
2. **不要放快捷入口** — "伙伴""打卡""社区"已经在底部 Tab 栏，不要重复
3. **不要用大色块 Hero** — 不要蓝色渐变大卡片，保持白底+白卡片的干净风格
4. **不要用 Vant Tabs** — 首页是一个完整的滚动页面，不需要 tab 切换，所有内容按顺序展示
5. **每个区域之间间距统一** — section 之间 `margin-top: 24rpx`
6. **字号层级**：标题 28rpx，正文 26-27rpx，辅助 22-24rpx，时间 20rpx
